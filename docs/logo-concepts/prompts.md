## 使用建议

1. 同一个提示词在不同工具多试几次，结果差异很大
2. 满意后要求生成 **透明背景 PNG** 或 **SVG** 格式
3. 最终尺寸需要 128×128（favicon）和 512×512（高清版）
4. 图标颜色建议在深色（#0d1117）和浅色背景下都测试一下效果
